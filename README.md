## 📋 Descripción

Portfolio personal de Jean Dimter, desarrollado como proyecto final del bootcamp Desafío Latam. Este proyecto es un sitio web responsive que muestra experiencia profesional, proyectos, habilidades y formas de contacto.

## ✨ Características

- **Diseño Responsive**: Adaptado para dispositivos móviles, tablets y desktop
- **Interfaz Moderna**: Utiliza Bootstrap 5 para un diseño limpio y profesional
- **Navegación Intuitiva**: Menú fijo en la parte superior para fácil acceso
- **Iconografía**: Integración de Font Awesome para iconos profesionales
- **Tipografía Personalizada**: Fuente Raleway de Google Fonts para mejor legibilidad
- **Secciones Principales**: Acerca de mí, proyectos, habilidades y contacto

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura semántica
- **CSS3**: Estilos personalizados
- **Bootstrap 5**: Framework CSS responsive
- **Font Awesome 5**: Biblioteca de iconos
- **Google Fonts**: Tipografía Raleway

## 📁 Estructura del Proyecto

```
proyecto_final/
├── index.html              # Página principal

├── assets/
│   ├── css/

```

## 🚀 Cómo Usar

1. **Clonar el repositorio**
   ```bash
   git clone https://github.com/tu-usuario/proyecto_final.git
   ```

2. **Abrir en el navegador**
   - Navega a la carpeta del proyecto
   - Abre el archivo `index.html` en tu navegador preferido

3. **Editar contenido**
   - Modifica el archivo `index.html` para cambiar el contenido
   - Actualiza los estilos en `assets/css/estilos.css`
